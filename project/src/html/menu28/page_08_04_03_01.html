미사경정공원 위치

경기도 하남시 미사대로 505(신장동)

자가용 이용

성남 및 분당지역 수도권 제1순환고속도로 이용

성남대로에서 수도권제1순환선 구리, 상일 방면으로 진입 -> 상일IC에서 천호대교, 하남 방면으로 오른쪽 고속도로 출구 진입 -> 상일 IC에서 하남 방면으로 오른쪽 방향 -> 진등지하차도에서 ‘미사역’ 방면으로 왼쪽 지하차도 진입

강북 방면 세종포천고속도로(구리-포천) 이용

월드컵북로 방면 -> 성산2교에서 연희IC 방면으로 좌회전 -> 수내역, 내부순환로 방면으로 고가차도 오른쪽 옆길 -> 연희교차로에서 ‘내부순환로’ 방면으로 오른쪽 고가차도 진입, 도시고속도로 진입 -> 하월곡분기점교차로에서 ‘북부간선도로(구리·월릉IC)’ 방면으로 오른쪽 방향 -> 중랑IC에서 남구리IC 방면 오른쪽 고속도로 진입 -> 남구리IC에서 서울, 강변북로 방면으로 오른쪽 고속도로 출구 -> 아천 IC에서 구리암사대교, 올림픽대교 방면 -> 암사IC에서 강일IC 방면 -> 강일IC에서 서울·양양고속도로, 춘천, 덕소삼패 방면으로 고속도로 진입 -> 미사경정공원, 미사강변도시 방면으로 지하차도 오른쪽 옆길 -> 미사사거리에서 미사경정공원(후문) 방면으로 좌회전

강남 방면

올림픽대로 방면으로 도시고속도로 진입 -> 강일IC에서 서울·양양고속도로, 춘천, 덕소삼패 방면으로 고속도로 진입 -> 미사교차로에서 팔당대교, 하남시청, 양평 방면으로 오른쪽 고속도로 출구 -> 미사경정공원, 미사강변도시 방면으로 지하차도 오른쪽 옆길 -> 미사사거리에서 미사경정공원(후문) 방면으로 좌회전
