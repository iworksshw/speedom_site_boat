checkbox / radio
- form은 기본적으로 .frmGroup안에서 사용합니다.
- checkbox, radio는 .frmBox.maxCont 안에서 사용합니다.
- 라인 하나에 여러개를 꽉차게 넣을 때는 .frmBox.maxCont에서 .maxCont를 뺀 .frmBox만 사용합니다.
- 텍스트가 없을 경우 .iptLbl에 .noText를 추가한다.
- 크기가 다른 것 (미정)
- 기본 margin-right (미정)
- 너비가 고정일경우 (미정)
버튼형 checkbox / radio (미정)
- 사용방법은 위와 동일한 상태에서 .iptLbl 대신에 .iptLblBtn 사용합니다.
